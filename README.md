# Using a Custom Domain with React + Vite and Surge

If you want your React project to have a **custom domain** (instead of the Surge-generated `.surge.sh` domain), follow these steps.

---

## Step 1: Prepare Your Custom Domain
- Buy a domain from providers like **Namecheap, GoDaddy, or Google Domains**.  
- For example: `www.myreactapp.com`.

---

## Step 2: Add a CNAME File
1. Open your project folder.  
2. Go to the **`public`** folder 
3. Create a new file called: CNAME
